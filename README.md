## 📌 About Me
#### 교육 현장의 비효율을 기술로 혁신하는 **프로덕트 엔지니어**입니다.

✅ **실무 기반의 에듀테크(EduTech) 도메인 전문가**
- 2년간 코딩 학원 강사로 근무하며 학생과 교육자의 페인포인트(Pain-point)를 최전선에서 경험했습니다.
- NYPC, KOI 등 알고리즘 대회 지도 경험을 바탕으로, 단순한 코드 작성을 넘어 '학습자 중심의 시스템'이 어떻게 설계되어야 하는지 깊이 이해하고 있습니다.

✅ **AI 활용 및 끈질긴 문제 해결 능력 (In-house 서비스 개발)**
- 수동적인 학원 행정 업무를 자동화하기 위해 **Python/Flask 기반의 학생 관리 및 진도 시각화 서비스**를 직접 기획하고 배포했습니다.
- LLM(Codex 등)을 적극 도입하여 개발 생산성을 극대화하는 한편, AI가 처리하지 못하는 복잡한 비정형 데이터(JSON 파싱 등)와 예외 상황을 직접 끈질기게 디버깅하며 서비스 정합성을 높였습니다.

✅ **도구의 한계를 넘어서는 '메이커(Maker)'의 실행력**
- 환경의 제약에 좌절하기보다, 당장 가진 도구들을 활용해 눈앞의 문제를 해결하고 쓸모 있는 결과물을 만들어내는 과정을 사랑합니다.
- 웹 백엔드, AI 응용(OpenCV)부터 클라우드 환경까지, '문제 해결'이라는 목적을 달성하기 위해 필요한 기술이라면 주저 없이 학습하고 서비스에 적용합니다.
